WEB BROWSERS

A web browser (commonly referred to as a browser) is application software for accessing the World Wide Web. When a user requests a web page from a particular website, the web browser retrieves the necessary content from a web server and then displays the page on the user's device.


TOP 6 WEB BROWSERS

Google Chrome

Cross-platform web browser developed by Google. It was first released in 2008 and as of October 2021, Chrome has a 65.67% worldwide browser market share. It is available for Android, ChromeOS, iOS, Linux, Windows, and macOS.



Safari

Graphical web browser developed by Apple. It was first released in 2003 and as of October 2021, Safari has a 19.06% worldwide browser market share. It is available for iOS and macOS devices.



Microsoft Edge

Cross-platform web browser created and developed by Microsoft. It was first released in 2015, and the new Chromium-based Edge was released in 2020. As of October 2021, Edge has a 3.99% worldwide browser market share. It is available on Windows, macOS, Android, iOS, and Linux.



Mozilla Firefox

A free and open-source web browser developed by the Mozilla Foundation and its subsidiary, the Mozilla Corporation. It was first released in 2002. As of October 2021, Firefox has a 3.66% worldwide browser market share. It is available for Linux, macOS, Windows, Android, and iOS.



Samsung Internet

A mobile browser for smartphones and tablets developed by Samsung Electronics. Samsung Internet replaced the stock Android browser as the default on Samsung Galaxy devices in 2012. As of October 2021, Samsung Internet has a 2.81% worldwide browser market share. It is available for Samsung devices running Android and Tizen.



Opera

A multi-platform web browser developed by its namesake company Opera. It was first released in 1995, making it one of the oldest desktop web browsers still actively developed today. As of October 2021, Opera has a 2.36% worldwide browser market share. It can be used on Windows, macOS, Android, iOS, and Linux operating systems. The company released a gaming-oriented version of the browser called Opera GX in 2019.
